# MavrogBattleCry-Release

This repository hosts releases of the **MavrogBattleCry** World of Warcraft addon.

## Installation

Use the **[Mavrog Updater](https://github.com/Mavrag/mavrog-updater/releases/latest)** for automatic installs and updates.

Or download the latest zip from the [Releases](../../releases/latest) page and extract the `MavrogBattleCry` folder into your `_retail_/Interface/AddOns` directory.
